CuplIDE

The CuplIDE eclipse product provides Cupl functionality from within eclipse workbench.

Download CuplIDE Unzip and run.

Note: Use 7-Zip to unzip the file. Native Windows 10, same times give a "Unknown Publisher" warning.

CuplSDK eclipse plugin

The CuplSDK eclipse plugin provides Cupl functionality from within eclipse. Apart from "standard" Cupl functionality, the eclipse plugin contains a Cupl project nature and also adds a new Cupl perspective.The plugin makes use of the all core functionalities of the WinCupl.

Requirements

Eclipse IDE. Browse the list of packages, looking for ‘Eclipse IDE for Java Developers’ and install.

Atmel WinCUPL must be installed.

Installing CuplSDK Eclipse Plugin

1. Select Help > Install New Software...

2. In the Work with field, type https://TEU_USER.github.io/TEU_REPO/CuplSDK/update/ and click Add...

3. Select Cupl

4. Check the CuplSDK Feature checkbox and click Next. Then follow the dialogs to install the CuplSDK feature. Also, trust us and accept the unsigned plugin

5. Restart Eclipse. The installation is now complete

The CuplSDK Perspective

Once installed, open the CuplSDK Perspective by using the standard Window/Open Perspective/Other... Command and selecting "Cupl" from the list. The perspective will open 3 views:

* A "CuplSDK Layout" view to the left containing device layout

* A "CuplSDK Symbols" view to the bottom containing symbols of the generated file.

The CuplSDK Project Wizard

Once installed, create a Cupl project by using the standard File/New/Project... Command and selecting "Cupl" from the list. The wizard allows creating 3 templates project:

* A "ATF22V10" project based

* A "ATF750C" project based

* A "ATF15xx" project based
